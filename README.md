# 仓储管理系统简介 python456

## 项目概述

本系统是一个基于Python语言的仓库仓储管理系统，采用Vue2和Django框架，以MySQL作为数据库管理系统，实现了前后端分离的开发模式。

## 技术栈

- 后端：Python 3.8、Django
- 前端：Vue2
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 商品管理
- 仓库管理
- 订单管理

## 系统角色

- 管理员：负责系统的维护与管理
- 员工：负责日常仓库操作

## 运行环境

- Python 3.8
- Node.js（版本需高于14）
- MySQL

1. 安装配置Python 3.8
2. 安装配置MySQL
3. 安装配置Node.js（版本需高于14）
4. 克隆项目代码，进行前后端部署
